## 产品概要

BlockUp Puzzle 是一款面向休闲玩家的 H5 方块放置消除游戏，核心特性包括：

- 8x8 棋盘 + 37 种方块形状
- 行列消除 + 连击倍数系统
- 三种策略道具：炸弹 / 重力 / 刷新
- 金币经济 + 复活机制
- 纯前端，无需后端，支持 PWA 离线游玩

## 技术栈

- 原型基于单文件 HTML + Tailwind CDN + 原生 JavaScript
- 双击 `.html` 文件即可在浏览器中打开使用
